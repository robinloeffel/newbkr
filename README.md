## What is this?

NewBKR is an alternative frontend that will connect to your Interactive Brokers account through their OAuth API. It aims to provide a cleaner, more intuitive user experience than the official platforms.

## Quick Start

```bash
npm install

npm start
```

Then open [localhost:5173](http://localhost:5173) and authenticate with your IBKR account.
